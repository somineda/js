## 🔧 기술 스택

- HTML / CSS / JavaScript
- Bootstrap 5
- Node.js + Express
- JSON 파일 기반 데이터 저장
- Git + GitHub

---

## 🚀 실행 방법

# 1. 프로젝트 클론
git clone https://github.com/somineda/js.git

# 2. 폴더 이동
cd kream_admin_backend

# 3. 의존성 설치
npm install

# 4. 서버 실행
node server.js
